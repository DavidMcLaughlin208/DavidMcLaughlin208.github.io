B L E N D E R 3 D

Blender is a jack-of-all trades open source 3D modeling program

There are many autoCAD type programs out there all with various emphases and price points.

Other programs in the same genre as Blender are Maya, 3DS Max, Real Flow, and Houdini.

Blender Does:

So many things!!

  • 3D Modeling
  • Still Frame & Animation Rendering
  • Armature Rigging & Animation
  • Stylized or Physically Based Rendering
  • Material Nodes
  • Procedural Textures
  • Camera Tracking
  • Physics Simulations
    • Rigidbody
    • Softbody
    • Cloth
    • Fluid
  • Countless user made add-ons to provide more functionality

Pros of Blender:

  • It's Free
  • It's open source
  • It can do almost anything other paid programs can
  • There are many free tutorials and learning resources

Cons of Blender:

  • Development and feature integration can be slower than other programs
  • The user interface is known for being confusing and unintuitive
  • There is little user support (aside from other users)

Blender is Open Source

Blender is open source and there is always at least one dedicated, paid (by donation) employee, however, anyone can contribute. One of the ways Blender has made this easy is by adding functionality to write scripts in Python using the Blender API to perform simple to complex tasks.

On GitHub people can create repositories for the 'add-ons' they have created. You simply need to download the file they provide and tell Blender where to find the add-on.
