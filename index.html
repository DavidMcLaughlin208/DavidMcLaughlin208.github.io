Blender 3D

Blender is a jack-of-all trades open source 3D modeling program

There are many autoCAD type programs out there all with various emphases and price points.

Other programs in the same genre as Blender are Maya, 3DS Max, Real Flow, and Houdini.

Here are a few reasons why someone would choose Blender over these other programs:

  • It's Free
  • It's open source
  • It can do almost anything other paid programs can
  • There are many free tutorials and learning resources

Also, there are reasons someone would choose another program over Blender. Here are some downfalls:

  • Development and feature integration can be slower than other programs
  • The user interface is known for being confusing and unintuitive
  • There is little user support (aside from other users)

So what can Blender actually do?

So many things!!

  • 3D Modeling
  • Armature Rigging & Animation
  • Stylized or Physically Based Rendering
  • Material Nodes
  • Procedural Textures
  • Camera Tracking
  • Physics Simulations
    • Rigidbody
    • Softbody
    • Cloth
    • Fluid
  • Countless user made add-ons to provide more functionality
